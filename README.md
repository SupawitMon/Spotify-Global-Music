# Spotify Hit Predictor Pro

A Machine Learning deployment project that predicts **song popularity (0–100)** from Spotify-style audio features.

## Project Goal
This project demonstrates a complete end-to-end ML workflow:
- define a meaningful problem
- understand and clean the dataset
- perform EDA
- build and compare multiple models
- tune hyperparameters
- evaluate using proper metrics
- deploy a Streamlit web app
